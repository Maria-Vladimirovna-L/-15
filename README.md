# 🎲 Random Task Generator

**Автор:** Сидорина Алёна Евгеньевна
**Версия:** 1.0

## Описание программы

«Random Task Generator» — это простое приложение для борьбы с прокрастинацией. Программа случайным образом выбирает для вас задачу из предопределенного списка или позволяет добавить свою собственную.
*   **Генерация:** Кнопка выбирает случайное дело на сегодня.
*   **История:** Все сгенерированные задачи сохраняются в файл.
*   **Фильтрация:** Можно отобразить только задачи определенного типа (Учеба, Спорт, Работа).
*   **Расширение:** Вы можете добавить свои собственные задачи.

## Системные требования

*   **Python:** версия 3.7 или новее.
*   **Библиотеки:** `tkinter` (входит в стандартную библиотеку Python), `random`, `json`.
*   Установка зависимостей не требуется.

## Запуск приложения

1.  Убедитесь, что Python установлен.
2.  Скачайте файлы проекта.
3.  Запустите скрипт из терминала:
    ```bash
    python main.py
    ```
4.  Откроется окно приложения.

## Как пользоваться программой

### 1. Генерация случайной задачи
Нажмите кнопку **«Сгенерировать задачу»**. Приложение выберет случайное действие из списка и покажет его в верхней части окна. Задача автоматически добавится в историю.

### 2. Добавление собственной задачи
Нажмите кнопку **«Добавить свою задачу»**.
В появившемся окне введите:
*   **Название:** Что нужно сделать (например, «Полить цветы»).
*   **Тип:** Категория задачи (например, «Дом»).
Задача будет добавлена в общий пул для генерации и в историю.

### 3. Просмотр истории и фильтрация
В нижней части окна отображается список всех ранее сгенерированных задач.
Используйте выпадающий список **«Фильтр по типу»**, чтобы отобразить только задачи определенной категории (например, только «Спорт»).

| **Действие** | **Ввод / Действие** | **Ожидаемый результат** |
